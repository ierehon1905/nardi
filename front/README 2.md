## Deployment

First, create a `.env` file in the root of the project. It should look like this:

```sh
NARDY_FTP_HOST=...
NARDY_FTP_USER=...
NARDY_FTP_PASSWORD=...
NARDY_FTP_DESTINATION=...
```
