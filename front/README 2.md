## Deployment

First, create a `.env` file in the root of the project. It should look like this:

```sh
HAIFIVE_FTP_HOST=...
HAIFIVE_FTP_USER=...
HAIFIVE_FTP_PASSWORD=...
HAIFIVE_FTP_DESTINATION=...
```
